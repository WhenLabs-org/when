The toolkit
Everything your agent needs

Six purpose-built tools for modern developer workflows. Each one does one thing well.

velocity
MCP
Task timing and estimation for coding agents. Builds a local SQLite history so your agent learns its own speed and gives data-backed ETAs.
$ velocity_start_task · velocity_estimate
stale
MCP CLI
Detect documentation drift. Catches wrong file paths, dead links, phantom env vars, outdated npm scripts, and port mismatches in your docs.
$ when stale scan
envalid
MCP CLI
Validate .env files against a typed schema. Cross-env sync checks, secret leak scanning, and detection of undocumented env vars in code.
$ when envalid validate
berth
MCP CLI
Detect and resolve port conflicts. Scans .env, docker-compose, and package.json to find collisions before you hit EADDRINUSE.
$ when berth status
aware
MCP CLI
Auto-detect your stack and generate AI context files for Claude Code, Cursor, GitHub Copilot, and AGENTS.md — all from one config.
$ when aware init --targets all
vow
MCP CLI
Scan dependency licenses and validate against plain-English or JSON policies. Generate attribution files and find compliant replacements.
$ when vow scan
How it works
Install once. Use everywhere.

One package registers a single MCP server with 30+ tools. Works with Claude Code, Cursor, VS Code, and Windsurf. Also installs a standalone CLI.

1

Install

One command registers a single MCP server (whenlabs) and writes CLAUDE.md instructions. All six tools — including velocity — are available immediately as when <tool> subcommands and as MCP tools your agent calls automatically.

2

Use

Your agent calls the right tool at the right time. After a refactor, it runs stale scan to catch doc drift. Before a release, vow check validates licenses. Before starting dev, berth check finds port conflicts. Run when doctor for a unified health report.

3

Improve

Background scans run on a schedule and surface issues in your status line. Every timed task builds your velocity profile so your agent can estimate future plans. The whole toolkit compounds over sessions — zero maintenance.

Cursor

VS Code / Windsurf

npx @whenlabs/when install --vscode
npx @whenlabs/when install --windsurf
npx @whenlabs/when install --all      # All editors at once

CLI only (no MCP)

npm install -g @whenlabs/when

# All six tools available globally as: when stale / when berth / when vow / ...

CI/CD GitHub Actions

# .github/workflows/ci.yml
- name: WhenLabs checks
  uses: WhenLabs-org/when@main

# Or run directly: exits 1 on issues
npx @whenlabs/when ci
